+-------------+-----------------------------------------------------------+
| option      | description                                               |
+-------------+-----------------------------------------------------------+
| -01         | (default) filter based on filename                        |
+-------------+-----------------------------------------------------------+
| -02         | filename first then file type                             |
+-------------+-----------------------------------------------------------+
| -03         | allow find to automatically re-order the search based on  |
|             | efficient use of resources and likelihood of success      |
+-------------+-----------------------------------------------------------+
| -maxdepth X | search current directory as well as sub-directories X     |
|             | levels deep                                               |
+-------------+-----------------------------------------------------------+
| -iname      | search without regard of test case                        |
+-------------+-----------------------------------------------------------+
| -not        | return results that does not match the test case          |
+-------------+-----------------------------------------------------------+
| -type f     | search for files                                          |
+-------------+-----------------------------------------------------------+
| -type d     | search for directories                                    |
+-------------+-----------------------------------------------------------+
